In order to obtain the dataset that was used for this study, the following steps can be followed:

- Navigate to https://aylien.com/resources/datasets/coronavirus-dataset
- Fill in the required registration details and click on Request Access
- An email will be sent with the download link for the dataset
- The download link downloads the following file: "aylien_covid_news_data.tar.gz"
- This file unzips to a JSON lines file
- Refer to NOTEBOOK_README on how to extract fields
